| ε        | Empty String / Epsilon                             | 空串         | 长度为 0 的串。                      |

| DFA      | Deterministic Finite Automaton                     | 确定有穷自动机    | 无 ε 边，且每个状态对每个输入符号有唯一转移的有穷自动机。 |
| Dtran    | DFA Transition Table                               | DFA 转换表    | 子集构造中记录 DFA 状态在输入符号下转移到哪个状态。   |

## 2.1 语法分析：自顶向下技术  
  
| 英文简称   | 英文全称                                            | 中文名         | 概念简述                                      |
| ------ | ----------------------------------------------- | ----------- | ----------------------------------------- |
| BNF    | Backus-Naur Form                                | 巴科斯-诺尔范式    | 描述上下文无关文法的标准记法。                           |
| EBNF   | Extended Backus-Naur Form                       | 扩展巴科斯-诺尔范式  | 在 BNF 基础上加入可选、重复等简写形式。                    |
| CFG    | Context-Free Grammar                            | 上下文无关文法     | 四元组 `G=(V, Σ, P, S)`；产生式左部是单个非终结符。        |
| CSG    | Context-Sensitive Grammar                       | 上下文有关文法     | 产生式可依赖上下文，左部可含上下文符号。                      |
| PDA    | Pushdown Automaton                              | 下推自动机       | 与上下文无关语言等价的自动机模型。                         |
| LBA    | Linear Bounded Automaton                        | 线性有界自动机     | 与上下文有关语言相关的受限图灵机模型。                       |
| L(G)   | Language of Grammar G                           | 文法 G 的语言    | 从 G 的开始符号可推导出的全部句子集合。                     |
| LL(1)  | Left-to-right, Leftmost Derivation, 1 Lookahead | LL(1) 文法/分析 | 从左到右读入、构造最左推导、只看 1 个前瞻符号的预测分析。            |
| FIRST  | FIRST Set                                       | FIRST 集合    | 某文法符号串可能推导出的串的首终结符集合，必要时含 ε。              |
| FOLLOW | FOLLOW Set                                      | FOLLOW 集合   | 在某些句型中可紧跟在非终结符之后的终结符集合。                   |
| SELECT | SELECT Set                                      | SELECT 集合   | 某产生式在预测表中适用的输入符号集合；由 FIRST 和 FOLLOW 组合得到。 |
| EOF    | End Of File                                     | 文件结束符       | 输入结束标记，常用 `$` 表示。                         |
  
## 2.2 语法分析：自底向上技术  
  
| 英文简称    | 英文全称                                                        | 中文名           | 概念简述                                   |
| ------- | ----------------------------------------------------------- | ------------- | -------------------------------------- |
| LR(k)   | Left-to-right, Rightmost Derivation in reverse, k Lookahead | LR(k) 分析      | 从左到右扫描输入，构造最右推导逆序，使用 k 个前瞻符号。          |
| LR(0)   | LR with 0 Lookahead                                         | LR(0) 分析      | 不使用前瞻符号，仅凭项目集中点的位置决定动作的 LR 分析。         |
| CLOSURE | Closure of Item Set                                         | 项集闭包          | 若项期待某非终结符，则加入该非终结符所有候选产生式的点在开头项。       |
| GOTO    | GOTO Function                                               | GOTO 函数       | 描述项目集读入某文法符号后转移到的新项目集。                 |
| ACTION  | ACTION Table                                                | 动作表           | LR 分析表中由状态和终结符决定移进、归约、接受或报错的部分。        |
| GOTO    | GOTO Table                                                  | 转换表           | LR 分析表中由状态和非终结符决定新状态的部分。               |
| ACC     | Accept                                                      | 接受动作          | LR 分析表中表示输入被成功归约为开始符号的动作。              |
| SLR     | Simple LR                                                   | 简单 LR         | 在 LR(0) 项集基础上用 FOLLOW 集限制归约动作的分析方法。    |
| SLR(1)  | Simple LR with 1 Lookahead                                  | SLR(1) 分析     | 若按 SLR 方法构造出的表无冲突，则文法为 SLR(1) 文法。      |
| LR(1)   | LR with 1 Lookahead                                         | LR(1) 分析      | 项形如 `[A → α·β, a]`，每个项目携带精确上下文前瞻符号。    |
| LALR    | Look-Ahead LR                                               | 向前看 LR / LALR | 合并 LR(1) 项集核心以减少状态数，同时保留比 SLR 更强的分析能力。 |

  
## 3. 语义分析  
  
| 英文简称   | 英文全称                                   | 中文名           | 概念简述                                 |
| ------ | -------------------------------------- | ------------- | ------------------------------------ |
| SDD    | Syntax-Directed Definition             | 语法制导定义        | 在文法产生式上附着属性和语义规则，用于描述属性值如何计算。        |
| SDT    | Syntax-Directed Translation            | 语法制导翻译        | 在产生式体中嵌入语义动作，并在分析过程中执行。              |
| AG     | Attribute Grammar                      | 属性文法          | 无副作用的 SDD；用属性和规则形式化表达语义计算。           |
| RPN    | Reverse Polish Notation                | 逆波兰表示 / 后缀表示  | 运算符写在操作数之后，如 `a b c * +`，适合栈式求值。     |
| AST    | Abstract Syntax Tree                   | 抽象语法树         | 去除冗余语法细节，只保留与语义相关的核心结构。              |
| CST    | Concrete Syntax Tree                   | 具象语法树 / 具体语法树 | 完整反映文法推导过程，包含所有终结符和非终结符。             |
| YACC   | Yet Another Compiler-Compiler          | YACC 工具       | 经典语法分析器生成工具，常与 SDT 动作结合。             |
| JVM    | Java Virtual Machine                   | Java 虚拟机      | 可执行字节码的虚拟机，课件中用于说明栈式计算模型和 Java 编译执行。 |
| GCC    | GNU Compiler Collection                | GNU 编译器套件     | 工业级编译器集合，课件用作语义分析工程实现示例。             |
| HIR    | High-level Intermediate Representation | 高层中间表示        | rustc 中比 AST 更简洁、适合语义分析的表示。          |
| MIR    | Mid-level Intermediate Representation  | 中层中间表示        | rustc 中进一步降低层次、服务借用检查和后续优化的表示。       |
| DefId  | Definition Identifier                  | 定义标识          | rustc 中唯一标识定义的编号体系。                  |
| NodeId | Node Identifier                        | 节点标识          | rustc 中唯一标识 AST/HIR 节点的编号体系。         |
| NLL    | Non-Lexical Lifetimes                  | 非词法生命周期       | Rust 借用检查中不完全受词法块边界限制的生命周期分析。        |
| PEG    | Parsing Expression Grammar             | 解析表达式文法       | 带有确定性选择/回溯特征的文法形式；副作用需谨慎控制。          |
  
## 4. 中间代码生成  
  
| 英文简称         | 英文全称                               | 中文名             | 概念简述                           |
| ------------ | ---------------------------------- | --------------- | ------------------------------ |
| IR           | Intermediate Representation        | 中间表示            | 前端和后端之间相对独立的程序表示，支持优化和目标代码生成。  |
| AST          | Abstract Syntax Tree               | 抽象语法树           | 树形 IR，适合早期高层变换和语义分析。           |
| TAC          | Three-Address Code                 | 三地址码            | 每条指令最多涉及三个地址，形如 `x = y op z`。  |
| SSA          | Static Single Assignment           | 静态单赋值形式         | 每个变量只赋值一次，通过版本号和 φ 函数处理多路径定义。  |
| φ            | Phi Function                       | φ 函数            | SSA 中在控制流合流点根据前驱路径选择变量版本的伪指令。  |
| l-value      | Left Value                         | 左值              | 表示可存储位置/地址的表达式。                |
| r-value      | Right Value                        | 右值              | 表示表达式计算得到的值。                   |
| CFG          | Control-Flow Graph                 | 控制流图            | 以基本块为结点、可能执行顺序为边的图。            |
| DFG          | Data-Flow Graph                    | 数据流图            | 表示值依赖和数据流动关系的图。                |
| JVM Bytecode | Java Virtual Machine Bytecode      | JVM 字节码         | 栈式 IR 的代表，操作数隐含在操作数栈上。         |
| LLVM IR      | LLVM Intermediate Representation   | LLVM 中间表示       | 工业级中间表示，广泛用于优化和多后端代码生成。        |
| GIMPLE       | GIMPLE Intermediate Representation | GCC GIMPLE 中间表示 | GCC 中端常用的简化中间表示，常结合 SSA。       |
| RTL          | Register Transfer Language         | 寄存器传送语言         | GCC 后端较低层的中间表示，接近寄存器和机器指令。     |
| truelist     | True List                          | 真链              | 布尔表达式为真时应跳转的未定目标指令位置列表。        |
| falselist    | False List                         | 假链              | 布尔表达式为假时应跳转的未定目标指令位置列表。        |
| nextlist     | Next List                          | 下一链             | 语句执行完后应跳转到后继位置的未定目标指令列表。       |
| makelist     | Make List                          | 建链函数            | 创建只含一个跳转指令位置的列表。               |
| merge        | Merge                              | 合并函数            | 合并两个回填列表。                      |
| backpatch    | Backpatch                          | 回填函数            | 将某目标标号填入列表中所有未定跳转指令。           |
| nextquad     | Next Quadruple                     | 下一四元式位置         | 返回下一条将要生成的中间代码位置。              |

## 7. 机器无关优化

| GEN    | Generate Set                     | 生成集      | 基本块产生并传出到后续程序点的数据流事实集合。              |
| KILL   | Kill Set                         | 杀死集      | 基本块使其失效的数据流事实集合。                     |
| use    | Use Set                          | 使用集      | 基本块中先于任何定值被使用的变量集合。                  |
| def    | Definition Set                   | 定义集      | 基本块中被定值的变量集合。                        |
| e_gen  | Expression Generate Set          | 表达式生成集   | 基本块生成的可用表达式集合。                       |

| NAC    | Not A Constant                   | 非常量      | 常量传播数据流格中的状态，表示变量不是确定常量。             |
| SCC    | Strongly Connected Component     | 强连通分量    | 有向图中任意两点互相可达的最大子图。                   |
| LCA    | Lowest Common Ancestor           | 最近公共祖先   | 支配树中两个结点的最近公共祖先，对应最近公共支配者。           |
