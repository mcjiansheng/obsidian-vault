统计奇数个数
四进制数字为： 0, 1, 2, 3
其中奇数为 1, 3
为每个数字定义属性 $Num.odd$
表示：当前数字是否为奇数，若是 1 或 3 ，值为 1；若是 0 或 2，值为0

| 文法符号  | 属性        | 含义                  |
| ----- | --------- | ------------------- |
| $S$   | $S.cnt$   | 整个四进制小数中奇数数字个数      |
| $Q_1$ | $Q_1.cnt$ | 整数部分中奇数数字个数         |
| $Q_2$ | $Q_2.cnt$ | 小数部分中奇数数字个数         |
| $R$   | $R.inh$   | 当前已经累计的奇数数字个数       |
| $R$   | $R.syn$   | 最终返回给 (Q_1) 的奇数数字个数 |
| $Num$ | $Num.odd$ | 当前数字是否为奇数           |
$S \rightarrow Q_1$       `{ S.cnt = Q1.cnt; } `
$S→Q1​.Q2​$`{ S.cnt = Q1.cnt + Q2.cnt; }`

$Q_1​ → Num \space R$ 
` Q1 → Num { R.inh = Num.odd; } R { Q1.cnt = R.syn; }`

$R→Num \space R_1​$
`R → Num { R1.inh = R.inh + Num.odd; } R1 { R.syn = R1.syn; }`

$R→ϵ$
`R → ε { R.syn = R.inh; }`

$Q_2​→Num \space Q_{2,1}$
`{ Q2.cnt = Num.odd + Q2_1.cnt; }`
$Q2​→Num$
`{ Q2.cnt = Num.odd; }`

$Num →0$
`{ Num.odd = 0; }`
$Num →1$
`{ Num.odd = 1; }`
$Num →2$
`{ Num.odd = 0; }`
$Num →3$
`{ Num.odd = 1; }`

**将四进制小数转换为二进制字符串**
$4=2^2$

| 文法符号  | 属性         | 含义                |
| ----- | ---------- | ----------------- |
| $S$   | $S.bin$    | 整个四进制小数转换后的二进制字符串 |
| $Q_1$ | $Q_1.raw$  | 整数部分逐位转换后的二进制串    |
| $Q_2$ | $Q_2.raw$  | 小数部分逐位转换后的二进制串    |
| $R$   | $R.inh$    | 整数部分已累计的二进制串      |
| $R$   | $R.syn$    | 整数部分最终二进制串        |
| $Num$ | $Num.bits$ | 当前四进制数字对应的两位二进制串  |
