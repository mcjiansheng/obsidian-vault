```c
for (S1; !B; S2) S3
```
等价于
```c
S1;
while (!B) {
    S3;
    S2;
}
```
关键点：
```
B.true  应该跳到 S.next   // B 为真，则 !B 为假，退出循环
B.false 应该跳到循环体   // B 为假，则 !B 为真，进入循环体
```
### 语义规则
```
begin = newlabel()   // 条件判断入口
body  = newlabel()   // 循环体入口
step  = newlabel()   // 更新语句入口

对于产生式
S → for (S1; !B; S2) S3

的语义规则可以写为
begin = newlabel()
body  = newlabel()
step  = newlabel()

S1.next = begin

B.true  = S.next       // 注意：B 为真时，!B 为假，退出
B.false = body         // 注意：B 为假时，!B 为真，进入循环体

S3.next = step
S2.next = begin

S.code =
    S1.code
 || label(begin)
 || B.code
 || label(body)
 || S3.code
 || label(step)
 || S2.code
 || gen('goto' begin)
```

### 翻译：
```c
for (i = 0; i < n && sum < 1000; i = i + 1) {
    sum = sum + i;
}
```
考虑循环条件：
```c
i < n && sum < 1000
```
循环等价于
```c
i = 0;
while (i < n && sum < 1000) {
    sum = sum + i;
    i = i + 1;
}
```

#### 1. 拆分语法成分
```
S1: i = 0

B : i < n && sum < 1000

S2: i = i + 1

S3: sum = sum + i

对于布尔表达式：
B → B1 && B2

B1 → i < n
B2 → sum < 1000
```

#### 2.语法分析树与属性标签
```
L1: 条件判断入口
L2: 第二个条件 sum < 1000 的入口
L3: 循环体入口
L4: 更新语句入口
L5: 整个 for 语句的出口，即 S.next
```
语法分析树可以标成：
```
S0 [next = L5]
└── for ( S1 ; B ; S2 ) S3
    ├── S1: i = 0
    │   ├── S1.next = L1
    │   └── S1.code = "i = 0"
    │
    ├── B: B1 && B2
    │   ├── B.true  = L3
    │   ├── B.false = L5
    │   │
    │   ├── B1: i < n
    │   │   ├── B1.true  = L2
    │   │   ├── B1.false = L5
    │   │   └── B1.code =
    │   │       if i < n goto L2
    │   │       goto L5
    │   │
    │   └── B2: sum < 1000
    │       ├── B2.true  = L3
    │       ├── B2.false = L5
    │       └── B2.code =
    │           if sum < 1000 goto L3
    │           goto L5
    │
    ├── S2: i = i + 1
    │   ├── S2.next = L1
    │   └── S2.code = "i = i + 1"
    │
    └── S3: sum = sum + i
        ├── S3.next = L4
        └── S3.code = "sum = sum + i"
```
