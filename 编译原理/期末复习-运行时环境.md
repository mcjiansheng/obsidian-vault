```c
int m = 0;

void main() {
    int i = 100;
    int j = 20;
    m = sub(i, j);
    printf("m=%d", m);
}

int sub(int a, int b) {
    int res = 0;
    res = a - b;
    return res;
}
```
题目运行到12行，即`res = a - b;`
此时：
```c
a = 100
b = 20
res = a - b = 80

```
因为res未return 所以 m 仍然是 0

#### 1. A、B：代码区
A、B 位于代码区，不是变量存储单元，而是程序指令所在区域。
所以可以写成源代码层面的指令：

|位置|内容|
|---|---|
|A|代码：`int i = 100;`|
|B|代码：`int j = 20;`|

#### 2.C：静态数据区
```c
int m = 0;
```
全局变量具有静态存储期，放在静态数据区。
全局变量属于静态分配，编译时即可确定存储位置；而局部名字通常在栈中分配。
